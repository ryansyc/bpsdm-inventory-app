- install Laravel, composer

- install dependency dengan cmd: 
  composer install
  npm install 

- bikin database namanya: "bpsdm_inventory_app" di MySQL
  (bila mau ganti nama database, ubah juga DB_DATABASE di .env)

- cmd: php artisan migrate

- cmd: php artisan db:seed

- cmd: php artisan icons:cache

- cmd: php artisan serve atau npm run dev

- buka "http://127.0.0.1:8000" di browser
